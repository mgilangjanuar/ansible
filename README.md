Phabricator (Ansible + Docker)
==============================

Phabricator installer with Docker and Ansible.

## Requirements
 - Docker

## How to build and run
 - Run Docker
 - Run `docker build -t phabricator`
 - Run `docker run -d -p 80:80 phabricator`
